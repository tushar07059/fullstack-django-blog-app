# fullstack-django-blog-app
A full stack blog web application built using Python and Django.  The project allows users to create and manage blog posts with image uploads.  It demonstrates Django's core concepts including models, views, templates, forms, static files, and media handling using an SQLite database.
## Features

- User Authentication (Login, Register, Logout)
- Create, Update and Delete Blog Posts
- Image Upload Functionality
- Django ORM with SQLite Database
- Responsive UI using HTML and CSS
